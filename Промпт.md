Нужно написать Ruby on Rails приложение, которое будет работать в качестве бэка для приложения на планшете.

Модель данных:

Модель Провинция
- string Название
- belongs_to Страна

Модель Страна
- string Название

Модель Должность
- string Название
- belongs_to Страна

Модель Тип Действия
- string Тип (Варианты: prince, noble) 
- string Название
- string Параметры показа (Варианты: C, P, C2, PF)
- string Результат при успехе
- string Результат при неудаче

Действие
- belongs_to Должность
- belongs_to Тип действия
- belongs_to Страна
- belongs_to Вторая Страна
- belongs_to Провинция
- integer Номер цикла

Параметры
- integer Текущий номер цикла

Seeds:

Должности:
Маршал
Государь
Управляющий
Тайный советник
Епископ
Инженер
Дипломат

Провинции и страны в файле 
countries.csv

Типы действий в файле
roles.csv
Из этого файла загрузить каждую должность в отдельную страну. В Великое княжество Литовское и в Русь.

Нужны следующие экшены API (используй api.rsb):
1. Вывод в апи всех должностей вместе с их действиями в определённой стране.
2. Вывод Типа действия вместе с тремя списками в зависимости от страны C: Все страны, кроме C, провинции страны C, провинции всех стран, кроме C.
3. CRUD для всех моделей
4. Совершение действия должностью - сохраняет новое действие в модель Действие. В зависимости от типа действия заполняются по-разному поля:
- Тип C: Заполняется только Страна
- Тип P и PF: Заполняется только Провинция
- Тип C2: Заполняются Страна и Вторая Страна
5. Вывод совершённых действий в текущем цикле в определённой Стране вместе с типами действий.
6. Смена цикла

В приложении должны быть следующие экраны:

1. Экран выбора страны - Русь или Великое княжество Литовское. 
2. Экран выбора с двумя кнопками: "Совершить действие" и "Посмотреть совершённые действия".

В случае выбора кнопки "Совершить действие"
1. Появляется экран выбора должности в выбранной в п.1 стране. Сортировка по алфавиту
2. После выбора должности появляется список Типов действий (назови это "Действия") для должности. Сортировка по алфавиту 
3. При выборе Типа действия появляется экран с его свойствами:
- Название действия
- Результат при успехе
- Результат при неудаче
- В зависимости от типа действия есть выпадашки для выбора (во всех Сортировка по алфавиту):
Тип C: Заполняется только Страна
Тип P: Выбор Провинции страны, выборанной в п.1
Тип PF: Выбор Провинции всех стран, кроме выбранной в п.1
Тип C2: Две выпадашки выбора страны: Страна и Вторая Страна
- Кнопка "Совершить действие" с мягкой валидацией (если не выбраны нужные выпадашки - появляется окно "Не заполнено ... Точно отправить?"

В случае выбора кнопки "Посмотреть совершённые действия"
Здесь отображаются действия, которые были совершены в текущем цикле в порядке их совершения и выбранные параметры. Каждое действие можно пометить как прочитанное. Прочитать действие можно только один раз. Нужно отмечать прочитанные на бэке.
